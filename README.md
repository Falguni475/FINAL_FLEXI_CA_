# Online Voting System

A full-stack **Online Voting System** built with **React** (frontend) and **Node.js + Express** (backend) using **MongoDB**.  
Users can **register, login, vote once**, and view results on a dashboard.  


---

## Features

- **Frontend (React):**
  - Home, Login, Register, Vote, Dashboard pages
  - Axios for API calls to backend
  - JWT token stored in localStorage
  - Colorful and interactive UI

- **Backend (Node.js + Express):**
  - User registration and login with JWT authentication
  - Voting routes, connected to MongoDB
  - Only one vote per user
  - RESTful API design
